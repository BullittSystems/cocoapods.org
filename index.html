      CocoaPods is built with Ruby and is installable with the default Ruby available on OS X. We recommend you do this.

      Using the default Ruby install will require you to use sudo when installing gems.

      $ sudo gem install cocoapods

      Search for pods (above). Then list the dependencies in a text file named Podfile in your Xcode project directory: 

          $ edit Podfile
    platform :ios
    pod 'JSONKit',       '~> 1.4'
    pod 'Reachability',  '~> 3.0.0'

      Now you can install the dependencies in your project:

      $ pod install

      Make sure to always open the Xcode workspace instead of the project file when building your project:

      $ open App.xcworkspace

      Now you can import your dependencies e.g.:

      #import <Reachability/Reachability.h>

      We’re developing CocoaPods on GitHub. There’s a repository for the CocoaPods tool and one for all the Podspecs. It’s easy and really gratifying to contribute patches or pods, for a lot of people it's their first OSS pull request.

      Follow CocoaPods on Twitter to get up to date information about what’s going on in the CocoaPods world.

      Please report any problems with CocoaPods to the CocoaPods issues page.
